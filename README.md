# Weather App
 https://priyarajawat188.github.io/Weather-App1/
 [Live Demo]( https://priyarajawat188.github.io/Weather-App1/)
A sleek and responsive Weather Application built using **Vanilla JavaScript** and **OpenWeatherMap API**. This project demonstrates the use of asynchronous programming,
dynamic DOM manipulation, and real-time data fetching.

# Features

- **Real-time Weather Data:** Fetches current temperature, humidity, wind speed, and weather conditions.
- **5-Day Forecast:** Displays the upcoming weather forecast for the searched city.
- **Search Functionality:** Users can search for any city globally.
- **Responsive Design:** Optimized for a great user experience on mobile and desktop.
- **Dynamic UI:** Background blur effects and interactive elements using CSS3.

# Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **API:** [OpenWeatherMap API](https://openweathermap.org/api)
- **Icons:** Material Symbols (Google Fonts)
